The ML component is based on a graph neural network (GNN) architecture, which is trained on a large dataset of SQL queries and their corresponding semantic representations. The formal methods component uses a satisfiability modulo theories (SMT) solver to verify the correctness of generated queries.

* **Architecture:** GNN + SMT solver

* **Performance metrics:** Query accuracy, query execution time, and semantic correctness
